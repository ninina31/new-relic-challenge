## new-relic-challenge
Frontend technical challenge for New Relic recruitment process
SPA with Vanilla JS
Webpack as Module budler
Jest.js for Unit testing

### Usage

Instalation
```
npm i
```

Generate bundle
```
npm run build
```

Use project with server
```
npm run start
```

Unit testing
```
npm run test
```
